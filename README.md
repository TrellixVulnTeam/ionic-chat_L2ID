# GChat #

## Real time chat App using Ionic 3 & Angular 5 ##

This is a simple real time chat App made using Ionic 3 & Angular 5.

### App features ##

1. Chat anonymously just by putting name

2. Get older chat history with date & time

3. Delete all chats
